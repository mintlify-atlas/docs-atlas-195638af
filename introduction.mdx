title: React Native HealthKit
description: TypeScript-first React Native bindings for Apple HealthKit with full type safety and Promise support
---

React Native bindings for HealthKit with full TypeScript and Promise support covering about any kind of data. Keeping TypeScript mappings as close as possible to HealthKit - both in regards to naming and serialization.

<Note>
  This library is iOS-only and requires a physical device or simulator running iOS. HealthKit is not available on Android. For Android alternatives, see [react-native-health-connect](https://github.com/matinzd/react-native-health-connect/).
</Note>

## Supported data types

| Data Types                  | Query | Save  | Subscribe | Examples                               |
| ----------------------------|:------|:------|:----------|:---------------------------------------|
| 100+ Quantity Types         | ✅    | ✅    | ✅       | Steps, energy burnt, blood glucose etc.. |
| 63 Category Types           | ✅    | ✅    | ✅       | Sleep analysis, mindful sessions etc.. |
| 75+ Workout Activity Types  | ✅    | ✅    | ✅       | Swimming, running, table tennis etc..  |

| Clinical Records            | ⚠️    | ❌    | ⚠️       | Lab results in FHIR JSON format  |
